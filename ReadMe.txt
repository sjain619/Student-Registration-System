
As we developed this project as web application you will need server like tomcat and eclipse to easily run this project.

Steps:

1. run script proj2_demo_tables_script_2018.sql
2. run DBPROJECT-package.sql
3. run triggers.sql;
4. run logSeq.sql;
5. unzip the zip file at one location and you will find one project "student-registration".
6. Import the project into eclipse and right click on the project -> Run as -> run on server.
